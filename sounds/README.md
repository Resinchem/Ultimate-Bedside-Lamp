## Sound Readme
This folder contains various sounds you can download and use as potential alarm sounds by copying to your microSD card.  Information on preparing the SD card and how to add or substitute your own MP3 files as alarm sounds, see the [Build Guide](https://resinchemtech.blogspot.com/2026/04/ultimate-bedside-lamp.html) and the [Setting Up Alarm Sounds](https://resinchem.github.io/Ultimate-Bedside-Lamp/sounds.html) topic of the firmware guide.
