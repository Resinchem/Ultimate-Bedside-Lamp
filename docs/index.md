<div style="font-size: 18px; text-decoration: underline;">Hardware Substitutions</div>
<p>The firmware is written for a very specific set of hardware components.  The device design was based on my particular needs and wants. This is covered in more detail in the written guide, but any substitutions of hardware will very likely require corresponding firmware code changes.</p>
<p>Unfortunately, I simply do not have the bandwidth to create, maintain and upgrade multiple versions of the firmware. If you wish to use a different display, ESP board, touch sensors, etc. then you will need to fork or clone this repository, make the required changes and then compile your own version of the firmware.  See the section on Modifying the Firmware for more information.
<br><br>
<i>Please do not open issues requesting alternate hardware or inquiring if a particular hardware component will work!  Issues opened that request or utilize other hardware will be closed.</i><br>
<div style="font-size: 18px; text-decoration: underline;">Opening Issues</div>
This repo has both an issues section and a discussion section.  Please reserve opening an issue to things related to <i>issues related to firmware errors or that are not functioning as expected!</i><br><br>
If you have enhancement requests, questions about how to use the system or maybe think it <i>might</i> be firmware related, please post in the discussion area. If it is determined to be a firmware issue, I will move your discussion topic to an open issue.
