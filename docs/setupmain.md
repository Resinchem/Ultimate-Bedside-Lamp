Once the system has been onboarded and the controllers are speaking to one another, you can begin to configure the system for your particular needs, select a time and weather source and more.<br><br>

- [Web App Overiew](/webapp.md) - Covers accessing the web app, differences between default and active settings, etc. 
- [Main Display](/display.md) - Using and configuring the primary display, including the touch features
- [Auto-Dimming](/autodim.md) - Configuring Auto-Dimming for the Display 
- [Lights](/lights.md) - Setting and controlling the RGBW Bulb and LED Strip light sources
- [Clock and Time](/time.md) - Selecting and setting a time source. Changing the clock display font or color
- [Temperature and Weather](/weather.md) - Select and configure a source for outdoor temperature/weather
- [Touch Sensors](/touchsensors.md) - Configure and select features of the touch sensors

It is important to understand that the system has two "sets" of settings:

- _Default Settings_: Those saved settings and configuration values that are loaded when the system first starts up or reboots.
- _Active Settings_: Settings values that are currently is use.

This is covered in more detail in the Web App Overview.
