Prior sections mostly covered the initial configuration and setup of the system.  This section is going to focus on normal interaction with the system.  Setting and responding to alarms are covered in detail under the [Alarm](/alarmmain.md) sections so this topic will exclude dealing with alarms.

Before proceeding, assure you understand the different between **DEFAULT** settings loaded at boot time abnd **ACTIVE** settings which reflect the current state of the system.  When interacting with the system using the methods covered in this section, you are only changing the system's **ACTIVE** values.  This means that any changes made to value or states will be lost when the system restarts or reboots.  For more information, see the discussion of default vs. active settings within the [Web Application Overview](/webapp.md) topic.

General System Use will cover the following:

- [Controlling The Lights](/lightcontrol.md) - (both the RGBW light bulb and LED strip)

- [Managing the Display Brightness](/dispbrightness.md)

- [Using the Special Controller Commands](/commands.md)

Each of the above sections details the various methods of routinely controlling and interacting with the components of your project.
