# Installing Firmware Updates
<div align="center">

![firmwareupdates_01](images/firmwareupdates_01.jpg)
</div>

From time to time, new versions of the firmware may released.  This usually contain fixes for newly discovered bugs, new features or occasionally for new hardware.  This section covers how to obtain the last versions of the firmware, how the updates are organized and how to apply these updates to your own system.
