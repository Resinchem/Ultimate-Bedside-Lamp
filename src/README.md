## Source Code Files
The above folders contain the source code files.  They are divided between the files needed for the **PRIMARY** controller and the **DISPLAY** controller.  

- These are NOT interchangeable.  Installing the wrong firmware onto the wrong controller will turn your 
system into a dark, silent brick!
- You do not need the source files to just install the firmware.  If you want to install the firmware onto your ESP32, just download and use the .bin files found in the Asset section of the [latest release](https://github.com/Resinchem/Ultimate-Bedside-Lamp/releases/latest).
